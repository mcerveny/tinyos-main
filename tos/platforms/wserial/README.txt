README for wserial platform
Author/Contact: Martin Cerveny

Description:

The wserial platform is based on ZIGBIT/MESHBEAN (Atmel RF230 + Atmel ATmega1281) with EUI chip (Microchip 25AA02E48), 
flash (Atmel AT45DB041D) and 2 serial ports (ST3232). Optimalized to DIN rail case.
